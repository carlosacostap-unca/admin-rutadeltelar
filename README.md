## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## OpenSpec

This project uses [OpenSpec](https://github.com/Fission-AI/OpenSpec) for spec-driven changes.

```bash
npm run openspec -- list
npm run openspec -- list --specs
```

Codex OpenSpec skills are available under `.codex/skills`. Start a proposed change with `/opsx:propose "your idea"`, then apply it with `/opsx:apply` when the artifacts are ready.

## End-to-End Tests

This project uses Playwright for browser tests.

```bash
npm run playwright:install
npm run test:e2e
```

The E2E runner starts the Next.js dev server automatically on port `3100`. Set `PLAYWRIGHT_BASE_URL` to run tests against an already running environment instead.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
